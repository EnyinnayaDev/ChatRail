# frontend-chat-sandbox — Frontend Dev 2

**Language:** HTML, CSS, React

The WhatsApp-style chat sandbox — the demo-day centerpiece. Judges type or
record a voice note as the "customer" here.

Full brief and wireframe: `/docs/5_Frontend_Dev2_Chat_Sandbox.docx`

## Getting started

Nothing's scaffolded yet — start your project here, e.g.:

```bash
npm create vite@latest . -- --template react
npm install
```

Build against a mock version of the AI Engineer's JSON contract first (see
their brief doc) so you're not blocked waiting on their endpoint to be live.

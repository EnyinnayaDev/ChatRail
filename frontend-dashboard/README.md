# frontend-dashboard — Frontend Dev 1

**Language:** HTML, CSS, React

The Merchant Dashboard — kanban-style board with columns for each order
status, live-updating via WebSocket.

Full brief, wireframe, and endpoint list: `/docs/4_Frontend_Dev1_Merchant_Dashboard.docx`

## Getting started

Nothing's scaffolded yet — start your project here, e.g.:

```bash
npm create vite@latest . -- --template react
npm install
```

Build against `/database/seed.sql` data shapes first (via mock JSON) so
you're not blocked waiting on Backend Dev 1's API to be live.
